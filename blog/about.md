This blog is the **learning companion** to the
[A365 Governed Pro-Code Agent Starter](../README.md) repository. The repo
itself is the source of truth — runnable code, scripts, and reference docs.
The blog re-frames that material as a guided reading path so newcomers can
build a mental model before diving into source.

## How the posts map to the repo

| Post | Canonical doc in repo |
| --- | --- |
| What is Agent 365? | [`docs/learning-guide.md`](../docs/learning-guide.md) |
| Architecture & design | [`docs/design.md`](../docs/design.md) |
| Setup walkthrough | [`docs/setup-walkthrough.md`](../docs/setup-walkthrough.md) |
| Code walkthrough | [`docs/code-walkthrough.md`](../docs/code-walkthrough.md) |
| Security Blueprint policy | [`docs/blueprint-policy.md`](../docs/blueprint-policy.md) |
| Evidence: multi-instance inheritance | [`docs/evidence/multi-instance-inheritance.md`](../docs/evidence/multi-instance-inheritance.md) |
| Evidence: teardown → setup round-trip | [`docs/evidence/round-trip.md`](../docs/evidence/round-trip.md) |

If a post and a doc disagree, **the doc wins** — open an issue or PR against
the doc and the post will be re-synced.

## Try it yourself

Every post ends with a **Try it yourself** section. Those snippets assume you
have the prerequisites listed in the
[root README](../README.md#prerequisites) and that you have run the
[Quickstart](../README.md#quickstart) at least once.
