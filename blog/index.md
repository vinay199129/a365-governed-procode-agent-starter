Welcome. This is a blog-style learning series built around the
[A365 Governed Pro-Code Agent Starter](https://github.com/) — a working
reference for building **pro-code agents** that plug into
**Microsoft Agent 365 (A365)** as the enterprise governance layer.

Each post is a self-contained read with:

1. **Why it matters** — the problem the chapter solves.
2. **Key concepts** — short, linkable bullets.
3. **Try it yourself** — exact commands and files in the repo.
4. **Go deeper** — a link to the canonical doc that the post is based on.

Start at the top of the post list and work down — the series is ordered from
"what is A365" → architecture → setup → code → policy → evidence.

> **New to the repo?** Clone it first:
>
> ```pwsh
> git clone <this-repo-url>
> cd a365-governed-procode-agent-starter
> ```
>
> Then follow the [Quickstart in the root README](../README.md) before reading the
> code-walkthrough post.
