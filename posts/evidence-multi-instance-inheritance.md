Claim: every agent instance inherits its blueprint posture byte-for-byte.
Here is the captured proof — two instances, one blueprint, scope-set diff.

<!-- truncate -->

## What was tested

1. Provision instance **A** via the standard setup script.
2. Provision instance **B** against the *same* blueprint via
   `scripts/provision-second-instance.ps1`.
3. Read both instance SPs' delegated scope sets via Microsoft Graph.
4. Diff the scope arrays.

**Result: identical sets** — same Graph scopes, same Work IQ MCP scopes,
same `consentType`. No drift, no per-instance override path observed.

## Key concepts in five bullets

- **Inheritance is a provisioning-time projection** — the blueprint's scopes are copied onto each instance SP at create time, so every instance carries an identical (byte-for-byte) scope set with no per-instance config drift.
- **Reproducible** — anyone with the repo can reproduce the diff.
- **Pre-Frontier safe** — runs entirely against Entra; no A365 backend needed.
- **Tenant-portable** — only the IDs differ across tenants.
- **Foundation for compliance review** — hand this to your security reviewer.

## Try it yourself

```pwsh
pwsh -NoProfile -File scripts/setup-environment.ps1
pwsh -NoProfile -File scripts/provision-second-instance.ps1

# Resolve both instance SPs by display name (the script provisions them as
# "<AgentName> Identity" and "<AgentName>2 Identity").
$cfg    = Get-Content a365.config.json | ConvertFrom-Json
$prefix = ($cfg.agentIdentityDisplayName -replace ' Identity','').Trim()
$spA = az ad sp list --filter "displayName eq '$prefix Identity'"  --query '[0].id' -o tsv
$spB = az ad sp list --filter "displayName eq '${prefix}2 Identity'" --query '[0].id' -o tsv

# Diff the delegated grants (this is what `provision-second-instance.ps1` itself checks).
az rest --method GET --uri "https://graph.microsoft.com/v1.0/servicePrincipals/$spA/oauth2PermissionGrants" --query 'value[].{r:resourceId,s:scope}' -o json > a.json
az rest --method GET --uri "https://graph.microsoft.com/v1.0/servicePrincipals/$spB/oauth2PermissionGrants" --query 'value[].{r:resourceId,s:scope}' -o json > b.json
git diff --no-index a.json b.json   # expect: no diff
```
